# PDF座標エディタ

PDF上のフィールド座標をGUIでマッピングするための開発ツール。

## クイックスタート

```bash
pnpm install
pnpm dev
```

ブラウザで http://localhost:3000 にアクセス。

## 機能

- PDFファイルの読み込み・表示
- クリックで座標取得
- 矩形ドラッグでフィールド範囲選択
- フィールドのドラッグ移動
- 矢印キーで微調整（1pt / Shift+10pt）
- アライメント設定（左/中央/右、上/中央/下）
- JSONエクスポート/インポート
- グリッド表示

## 出力形式

```json
{
  "fields": [
    {
      "name": "field_name",
      "type": "text",
      "page": 1,
      "x": 100,
      "y": 200,
      "width": 150,
      "height": 30,
      "fontSize": 10,
      "align": "center"
    }
  ],
  "pdfDimensions": {
    "width": 595.2756,
    "height": 841.8898
  }
}
```

## 座標系について

- PDF座標系: 左下が原点 (0, 0)、Y軸は上方向が正
- A4サイズ: 595.28 x 841.89 ポイント

## 操作方法

| 操作 | 説明 |
|------|------|
| クリック | 位置を設定 |
| ドラッグ | 矩形選択（左下が座標） |
| マーカーをドラッグ | フィールドを移動 |
| 矢印キー | 1pt移動 |
| Shift+矢印キー | 10pt移動 |
| Delete | 選択フィールドを削除 |
